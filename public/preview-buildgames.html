Multi-Chain Architecture

GUNZscope operates a hybrid multi-chain architecture — reading game state from GunzChain (an Avalanche L1) while writing platform-native data to AVAX C-Chain.

This isn't a workaround — it's a feature. GunzChain is permissioned for contract deployment, so GUNZscope writes attestations, reputation badges, and trade intents to C-Chain. The result is cross-L1 composability within the Avalanche ecosystem — exactly the multi-chain thesis Avalanche is built on.

Solana provides the second read layer for NFTs minted on that chain, giving players a unified view regardless of where their assets live.

Valuation Waterfall

Not all valuations are created equal. GUNZscope distinguishes between market reality (actual sales data) and statistical proxies (estimates based on similar items).

Each NFT's value flows through a 6-tier waterfall — starting with the most specific data available and widening only when needed. Every valuation carries a confidence indicator so you always know what you're looking at.

The dual-track P&L system then applies two independent lenses: GUN appreciation (how the token itself has performed since acquisition) and market reference (what comparable items are trading for now).
